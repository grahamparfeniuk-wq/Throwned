# Throned

Throned is a gesture-first competitive arena for subjective media.

This is not a social feed, dashboard app, or passive browsing product.
The core product is emotionally meaningful competitive participation.

## Product Lens

- Battle sanctity over interface complexity
- Emotional investment over generic engagement
- Earned hierarchy over vanity metrics
- Cinematic restraint over noisy gamification

## Development

Install dependencies:

`npm install`

Run local development:

`npm run dev`

Build for production:

`npm run build`
