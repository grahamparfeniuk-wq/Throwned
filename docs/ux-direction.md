# UX Direction

Portrait:
- top vs bottom battle layout

Landscape:
- left vs right battle layout

VS badge:
- centered precisely on seam
- sleek diamond styling
- subtle glow/accent color

Metadata:
- hidden by default
- shown only on press-and-hold

Arena labels:
- centered
- elegant
- fade in/out
- clickable for arena search/jump

Leaderboard:
- draggable sheet
- hidden by default
- should feel physical and native

Animations:
- fast
- smooth
- tactile
- inertia-based when possible

The user should feel:
- tension
- competition
- momentum
- escalation

The interface should disappear behind the experience.
